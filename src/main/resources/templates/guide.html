Rules

There are a few rules to follow when creating and entering a user story.

  1. The user dictionary should be written in the following format: 'story word: explanation'. It is a single term. Subsequent terms should start with new lines.
  2. Remember not to enter any numbers numerically. Use full words, e.g.: instead of '4' write 'four'.
  3. Please do not use phrasal verbs.
  4. Please do not use phrasal verbs, and as few modal verbs as possible.
  5. Do not use any special characters except letters. The story is structured in such a way that their use is not necessary.
  6. No part of history can remain empty.
  7. Stick to the rule that sentence structure in each part should look like this: (person ->) action -> complement.
  8. Try not to use complex expressions in different parts of the story.
